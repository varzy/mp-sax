# MP-SAX

## 功能

- [x] 路由系统
- [ ] 增强原生的 Component 能力
- [ ] 使用 yml 替代 json 编写配置
- [ ] ESLint 检查代码
- [ ] Prettier 格式化代码

## 最佳体验

- 给微信开发者工具安装如下插件：
  - `yaml`
  - `eslint`
- 给微信开发者工具添加如下文件类型关联
  ```
  "files.associations": {
    "*.wxml": "xml",
    "*.wxss": "css",
    "*.wxs": "javascript"
  },
  ```
