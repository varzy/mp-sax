## 配置代码编辑器

- 给微信开发者工具安装如下插件：
  - `yaml`
  - `eslint`
- 给微信开发者工具添加如下文件类型关联
  ```
  "files.associations": {
    "*.wxml": "xml",
    "*.wxss": "css",
    "*.wxs": "javascript"
  },
  ```
